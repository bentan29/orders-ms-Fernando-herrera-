====== Prisma / Atencion prisma 7 tiene problemas asi que bajamos a prisma 6

    * Instalamos prisma / Prisma CLI (dev dependency)
        pnpm add -D prisma@6

    * Lo inicializamos
        pnpm prisma init

    * Instalacion prisma client
        pnpm add @prisma/client@6
        
    * Cremos el schema como querramos

    * Ejecutamos la migracion
        pnpm prisma migrate dev --name init


    ** Expandimos nuestro servicio que vamos a usar con prisma con 'extends PrismaClient implements OnModuleInit'
